# Library Management System

A simple C++ project for managing library books, members, and loans using procedural programming.

## Features
- **Books:** Add books and track available copies.
- **Members:** Register and manage library members.
- **Loans:** Borrow and return books with due date tracking.
- **Reports:** Generate overdue reports and calculate fines.
- **Storage:** Data is automatically saved to and loaded from `library_db.txt`.

## Technical Skills
- **Data Structures:** Structs and STL Vectors.
- **Algorithms:** Sorting using Templates and Predicates.
- **File I/O:** Reading/Writing data to text files.
- **Error Handling:** Using try-catch for system stability.
